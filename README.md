# PUBG 距离测量工具

## 功能
- 在PUBG地图上标记两个点，自动计算游戏内距离
- 支持多地图校准，切换地图自动加载对应校准值
- 语音播报测量结果

## 安装使用

### 方法一：直接运行Python脚本
```bash
pip install keyboard
python pubg_distance.py
```

### 方法二：双击启动
- 双击 `启动工具.bat`

### 方法三：桌面快捷方式
- 双击桌面上的「PUBG距离工具」快捷方式

## 使用说明

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + C` | 校准当前地图（选择两个相距1000米的点） |
| `T` | 标记点位（按两次计算距离） |
| `Ctrl + Shift + 1~9` | 切换地图 |
| `Ctrl + Q` | 退出程序 |

### 首次使用 - 初始化校准
1. 启动工具
2. 在PUBG中打开地图（按M键）
3. 按 `Ctrl + Shift + C` 进入校准模式
4. 按 `T` 键标记第一个点
5. 按 `T` 键标记第二个点（两点相距1000米）
6. 校准完成，自动保存

### 日常使用 - 测量距离
1. 按 `T` 键标记第一个点
2. 按 `T` 键标记第二个点
3. 自动语音播报距离

### 切换地图
- 按 `Ctrl + Shift + 1~9` 切换地图
- 语音播报地图名称
- 自动加载该地图的校准值

## 技术说明
- 使用Windows API获取鼠标坐标
- 校准值保存在 config.json，重启后自动加载
- 使用PowerShell语音合成播报距离

## 文件说明
- `pubg_distance.py` - 主程序
- `config.json` - 校准配置（自动生成）
- `requirements.txt` - Python依赖
- `启动工具.bat` - 一键启动
- `启动.vbs` - 后台启动（无窗口）
